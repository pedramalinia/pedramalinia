Projects

Telerehabilitation platform named Therapo

It's a professional platform for telerehabilitation, created and developed by myself. 
It's written in Python programming language and Django framework. 
In this platform therapists and clients can have online therapy sessions.
It has a lot of features like simple UI, simple client management, intelligent task builder, intelligent assessment tools, simple client's case management, therapy calendar and etc.
Website URL: Therapo

Speech generator device named TAG

It's an augmentative and alternative communication (AAC) high tech tool.
It's the very first device built in Farsi language.
In this device there are 3000 words include verbs, nouns and prepositions.
It can be used by patients that has communication problems. It can help them to make sentences and then tells the sentence to others verbally.

Persian Auditory Comprehension Test

It is a paper-based test for auditory comprehension assessment.
It has 20 plastic pieces in five different colors and in two size.
It can be used for evaluating auditory comprehension in people with aphasia.

Software version of Persian Auditory Comprehension Test

It is software version of Persian Auditory Comprehension test.
It is written in Python programming language and PyQt framework.
It can run the whole automatically and shows the final score.
